# IMPORTANT: Add AI API Keys to .env

Please add the following lines to your `.env` file:

```env
# GitHub Models (FREE - GPT-4o access)
GITHUB_MODELS_TOKEN=your_github_personal_access_token_here

# DeepSeek API
DEEPSEEK_API_KEY=your_deepseek_api_key_here
DEEPSEEK_BASE_URL=https://api.deepseek.com

# Gemini API
GEMINI_API_KEY=your_gemini_api_key_here
```

After adding these, restart your Laravel server for the changes to take effect.
