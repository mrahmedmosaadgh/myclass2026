# Sound Files for Behavior Manager

This directory contains sound files for the behavior management system.

## Files needed:
- `positive-chime.mp3` - Happy sound for positive behaviors
- `positive-chime.wav` - Happy sound for positive behaviors (fallback)
- `gentle-reminder.mp3` - Gentle sound for behaviors that need practice
- `gentle-reminder.wav` - Gentle sound for behaviors that need practice (fallback)

## Note:
If these files are not present, the system will automatically generate sounds using the Web Audio API as a fallback.

## Recommended sounds:
- Positive: Light chime, bell, or happy musical notes
- Negative: Gentle reminder tone, soft beep (not harsh or scary)
